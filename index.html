HTML

  • The head element contains information about the webpage.
  • The body element represents the visible content shown to the user.
  • DOCTYPE is a decleration which tells the browser what document type it should expect.
  • HTML lang tells the browser the language for the webpage.
  • title defines the title for the webpage.
  • style defines the CSS style associated with a webpage.
  • base defines the base URL for a webpage.
  • link connects an external resource to the HTML document.
  • meta defines metadata such as a character set, description, keywords, author and viewport.
  • script embeds client-side scripts in the HTML document.

Git

  • git status: checks what branch we are currently on
  • git checkout -b branch-name: creates a new branch and switches to it
  • git add -A: stages all files
  • git commit -m "changes-you-made": saved record of changes made to a file within a repo
  • git pull: fetches or pulls changes from a remote repo to your local repo
  • git push: sends or pushes the changes you've made in your local repo to a remote repo like GitHub
